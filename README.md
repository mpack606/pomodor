# Pomodor

A modern Pomodoro timer application featuring task-specific time tracking.

**Developed using Antigravity.**

## Tech Stack

- **Core**: React
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Build Tool**: Vite

## Features

- **Timer**: Customizable Focus, Short Break, and Long Break intervals.
- **Task Tracking**: Track accumulated time spent on specific tasks.
- **Notifications**: Audio alerts and system notifications when the timer ends.
- **Persistence**: Auto-saves your tasks and progress to local storage.

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
